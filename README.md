# Institute of Digital Risk (IDR) Homepage

Responsive single-page brand website built for the IDR internship assignment.

This project includes:

- a complete homepage with required sections
- a custom logo system (icon + full lockup, SVG)
- a clean dark visual theme using orange/white/black accents
- vanilla HTML/CSS/JS only (no frameworks)

## Assignment Scope Covered

The implementation includes all required parts:

- Hero section with mission-focused headline and CTAs
- About section
- Service pillars (Academy, Innovation, Advisory)
- Community / audience section
- Contact section with working form interaction
- Sticky navigation with smooth section scrolling
- Responsive behavior for desktop and mobile
- Hover/focus states for interactive elements

## Project Structure

- `index.html`  
  Semantic page structure and section content.

- `styles.css`  
  Design system tokens, layout, responsive rules, interactions, and motion.

- `script.js`  
  Mobile nav toggle, reveal-on-scroll behavior, active nav highlighting, and form thank-you state.

- `assets/idr-logo-icon.svg`  
  Icon-only cube mark.

- `assets/idr-logo-full.svg`  
  Full logo lockup (icon + wordmark).

- `note.txt`  
  Short logo rationale (colour, symbolism, typography intent).

## Design Notes

- **Visual direction:** dark, modern, professional UI with orange accents.
- **Typography:** Manrope-based system for clean readability.
- **Brand motif:** geometric cube to represent structure, resilience, and layered risk.
- **Iconography:** replaced emoji placeholders with inline SVG icons for production-like quality.

## How to Run

1. Clone or download the repository.
2. Open `index.html` in a modern browser.

No build step, package manager, or server is required.

## Tech Stack

- HTML5 (semantic structure)
- CSS3 (Grid, Flexbox, custom properties, media queries)
- Vanilla JavaScript (DOM events + Intersection Observer)
